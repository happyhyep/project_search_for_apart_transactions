## 핵심기능
> #### 지역별 아파트 실거래가 검색 기능
+ 원하는 지역 선택, 검색
+ 해당 지역의 아파트 목록 확인
+ 원하는 아파트의 실거래가 확인

> #### 상세 정보 안내 기능
+ 네이버 부동산 사이트 하이퍼링크 연결
